# Diagrams

This directory is reserved for standalone diagram source files (e.g., Mermaid `.mmd` files, exported SVG/PNG diagrams) supporting the architecture and design documentation under [docs/](../docs/).

Inline Mermaid diagrams currently live directly within the relevant Markdown documents (see [docs/architecture.md](../docs/architecture.md)). Additional standalone diagram assets will be added here as the design evolves.
